
---

## Features

- ⚡ Fully asynchronous (`asyncio`)
- 🌐 HTTP client based on `aiohttp`
- 🔐 Authentication and session handling
- 📂 File and directory operations
- ⬆️ File uploads (including chunked / multipart uploads)

---

## Installation

The project is not yet published on PyPI.

### Install from source

```bash
git clone https://github.com/devbis/aioterabox.git
cd aioterabox
pip install -e .
```


## Requirements

- Python 3.9+
- aiohttp
- cryptography
